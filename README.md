## Introduction to Docker

Docker is a platform for building, running, and shipping applications in a consistent manner.

Reasons for:
- One or more files are missing.
- Software version mismatch on the target machine.
- Different configuration settings between machines (such as environment variables).

Containers are isolated environments for running an application. In contrast to Virtual Machines, containers do not need a full-blown OS while requiring lower resources (lightweight). 

Regarding the architecture of Docker, it uses a client-server architecture
